Olá! Como desenvolvedor Delphi Senior com mais de 25 anos de experiência, posso te ajudar com essa questão. Existem algumas maneiras de extrair o nome do método do evento OnClick de um botão no formulário:

1. Usando o Object Inspector:

  • Clique duas vezes no botão para abrir o Object Inspector.
  • Na guia "Events", localize o evento "OnClick".
  • Na propriedade "Procedure Name", você encontrará o nome do método que será chamado quando o botão for clicado.

2. Usando o código Delphi:

  • Você pode usar a propriedade "Name" do objeto TButton para obter o nome do botão.
  • Em seguida, você pode usar a função "FindComponent" para encontrar o objeto TButton no formulário.
  • Finalmente, você pode usar a propriedade "OnClick" do objeto TButton para obter o nome do método.

Exemplo de código:

```delphi var Btn: TButton; MethodName: string;

begin Btn := FindComponent('BtnMyButton') as TButton; MethodName := Btn.OnClick.Name; end; ```

3. Usando a diretiva {$IFDEF CompilerVersion}:

  • Você pode usar a diretiva {$IFDEF CompilerVersion} para verificar a versão do compilador Delphi.
  • Se a versão do compilador for superior a 2009, você pode usar a propriedade "MethodName" do objeto TButton para obter o nome do método.

Exemplo de código:

```delphi {$IFDEF CompilerVersion >= 2009}

var Btn: TButton; MethodName: string;

begin Btn := FindComponent('BtnMyButton') as TButton; MethodName := Btn.MethodName; end;

{$ENDIF} ```

Observações:

  • O nome do método é sempre o mesmo que o nome do evento, com a adição do sufixo "Click".
  • Se você estiver usando um evento diferente de OnClick, você precisará usar a propriedade correspondente para obter o nome do método.
