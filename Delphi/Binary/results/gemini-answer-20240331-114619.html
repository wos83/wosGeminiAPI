Configurando o Delphi 10 para depuração com o IIS

Para depurar seu aplicativo Intraweb com o IIS no Delphi 10, siga estas etapas:

1. Configurando o IIS:

  • Instale o IIS: Certifique-se de ter o IIS instalado no seu computador. Você pode baixar e instalar o IIS gratuitamente no site da Microsoft.
  • Crie um site no IIS: Abra o Gerenciador do IIS e crie um novo site. Especifique o caminho para o diretório onde o seu aplicativo Intraweb está instalado.
  • Configure o pool de aplicativos: No Gerenciador do IIS, selecione o pool de aplicativos que o seu site está usando. Na seção "Avançado", configure o "Carregar assemblies em processo" como "True".

2. Configurando o Delphi:

  • Abra o projeto Delphi: Abra o projeto Delphi que contém o aplicativo Intraweb que você deseja depurar.
  • Adicione o IIS como servidor web: No menu "Projeto", selecione "Opções". Na guia "Servidor Web", clique no botão "Adicionar". Selecione "IIS" na lista de servidores web e clique em "OK".
  • Configure o servidor web IIS: Na guia "Servidor Web", selecione o servidor web IIS que você criou anteriormente. Especifique o nome do site que você criou no IIS no campo "Nome do site".
  • Ative a depuração: Na guia "Depuração", marque a caixa de seleção "Ativar depuração".

3. Iniciando a depuração:

  • Pressione F9 ou clique no botão "Executar" na barra de ferramentas do Delphi. O Delphi iniciará o servidor web IIS e implantará seu aplicativo Intraweb.
  • Abra o navegador da web e navegue até o URL do seu aplicativo Intraweb. O aplicativo será executado no servidor web IIS e você poderá usar o depurador do Delphi para depurar o código.

Dicas:

  • Verifique se o firewall está configurado para permitir o tráfego HTTP para a porta que o IIS está usando.
  • Se você estiver usando o Windows 10, precisará desabilitar o "Prompt de Controle de Conta de Usuário (UAC)" para que o depurador funcione corretamente.
  • Para obter mais informações sobre como depurar aplicativos Intraweb com o IIS, consulte a documentação do Delphi.
