Função para verificar se um número é primo em Delphi:

```delphi function EhPrimo(Numero: Integer): Boolean; var Divisor: Integer; begin Result := True; if Numero <= 1 then Result := False else for Divisor := 2 to Trunc(Sqrt(Numero)) do begin if Numero mod Divisor = 0 then begin Result := False; Break; end; end; end;

//Exemplo de uso var Numero: Integer; begin Numero := 13; if EhPrimo(Numero) then Writeln(Numero, ' é primo.') else Writeln(Numero, ' não é primo.'); end; ```

Explicação:

  • A função EhPrimo recebe um número inteiro como parâmetro e retorna um valor booleano.
  • A variável Result é inicializada como True.
  • Se o número for menor ou igual a 1, ele não é primo, então Result é definido como False.
  • Caso contrário, um loop for é usado para iterar por todos os números de 2 até a raiz quadrada do número.
  • Dentro do loop, se o número for divisível por qualquer divisor, Result é definido como False e o loop é encerrado.
  • Se o loop terminar sem que Result seja definido como False, o número é primo.

Exemplo de uso:

  • A variável Numero é definida com um valor.
  • A função EhPrimo é chamada com a variável Numero como parâmetro.
  • Se a função retornar True, o número é primo.
  • Se a função retornar False, o número não é primo.

Observações:

  • Este algoritmo é eficiente para verificar se um número é primo.
  • Existem outros algoritmos mais eficientes, mas este é fácil de entender e implementar.
  • Este algoritmo pode ser facilmente adaptado para verificar se um número é um número primo de Mersenne ou um número primo de Sophie Germain.
