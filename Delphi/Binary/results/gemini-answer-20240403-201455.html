Abrir uma nova página HTML sem barra de título e URL

Opção 1: Usando target="_blank"

  1. Insira o código HTML abaixo em sua página original:

html <a href="pagina_nova.html" target="_blank">Clique aqui</a>

  1. Substitua "pagina_nova.html" pelo nome da sua nova página.

Opção 2: Usando JavaScript

  1. Insira o código HTML abaixo em sua página original:

html <a href="javascript:void(0);" onclick="abrirNovaPagina()">Clique aqui</a>

  1. Adicione o seguinte código JavaScript ao seu arquivo JavaScript:

javascript function abrirNovaPagina() { window.open("pagina_nova.html", "_blank", "width=400,height=300"); }

  1. Substitua "pagina_nova.html" pelo nome da sua nova página e personalize as dimensões da nova janela.

Opções para ocultar a barra de título e URL:

1. Usando CSS:

Adicione o seguinte código CSS ao seu arquivo CSS:

css .frame-invisivel { border: none; background-color: transparent; position: absolute; top: 0; left: 0; width: 100%; height: 100%; z-index: 1000; }

2. Usando JavaScript:

Adicione o seguinte código JavaScript ao seu arquivo JavaScript:

javascript var novaJanela = window.open("pagina_nova.html", "_blank"); novaJanela.document.body.style.border = "none"; novaJanela.document.body.style.backgroundColor = "transparent";

Observações:

  • Ao usar a opção JavaScript, você pode definir outras propriedades da nova janela, como largura, altura e posição.
  • Utilize o CSS para personalizar a aparência da nova página, como cor de fundo e tamanho da fonte.
  • Certifique-se de que a nova página esteja configurada para ser aberta em uma nova aba/janela.
  • Tenha cuidado ao usar essa técnica, pois ela pode ser considerada enganosa se não for usada com cuidado.
